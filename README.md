# AltaFishing Web

Sitio web multipagina para AltaFishing, canal de YouTube y comunidad de pesca desde orilla en Chile, con foco en la Region del Biobio.

## Paginas

- `index.html`: Inicio y panel de condiciones de pesca.
- `quienes-somos.html`: Historia, enfoque y modalidades de spinning.
- `videos.html`: Grilla de videos de YouTube.
- `productos.html`: Catalogo visual de cañas, carretes y señuelos.
- `mareas.html`: Tabla de mareas, reloj visual, oleaje y datos sol luna aproximados.
- `contacto.html`: Formulario visual y redes sociales.

## Archivos compartidos

- `styles.css`: paleta blanco + azul metalico, responsive y animaciones.
- `script.js`: videos, catalogo filtrable, menu movil, formulario visual, olas animadas y dashboard de mareas.
- `img/altafishing.png`: logo oficial usado en la barra superior.

## Como verlo

Abre `index.html` en el navegador o usa la extension Live Server de Visual Studio Code.

## Edicion rapida

- Cambia los embeds de YouTube en el arreglo `videos` dentro de `script.js`.
- Cambia productos, textos e imagenes en el arreglo `products` dentro de `script.js`.
- Ajusta colores principales en `:root` dentro de `styles.css`.
- La ubicacion se intenta pedir automaticamente en Inicio y Mareas; los botones quedan como respaldo manual.
